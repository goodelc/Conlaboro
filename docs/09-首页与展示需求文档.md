# 09 — 首页与展示需求文档

> **模块**: 品牌落地页 / 项目浏览 / 公社原则展示 / 成果展厅 / 个人主页  
> **关联页面**: `HomePage`, `ShowcasePage`, `ProfilePage`  
> **核心价值传递**: 这三个页面承载 Conlaboro 的品牌故事、产品发现、成就展示  

---

## 1. 功能概述

本模块覆盖面向**访客和已登录用户**的三类展示型页面：

| 页面 | 核心目标 | 目标用户 | 设计情感 |
|------|----------|----------|----------|
| **HomePage** | 品牌故事 + 项目发现 | 访客/新用户 | **激励性** — "每个人都可以创造" |
| **ShowcasePage** | 已完工项目成果展示 | 所有人 | **荣誉感** — "看，这就是我们做的" |
| **ProfilePage** | 个人能力与贡献档案 | 所有人 (公开) | **信任感** — "这个人靠谱" |

这三个页面共同构成 Conlaboro 的「门面」— HomePage 吸引进来，ShowcasePage 展示成果，ProfilePage 建立个人信誉。

## 2. HomePage — 品牌落地页 `/` 或 `/home`

### 2.1 设计哲学

HomePage 不是传统的「功能导航页」，而是一个**品牌故事叙述页」。它的目标是让访客在 30 秒内理解：
1. 我们是谁（公社理念）
2. 这里能做什么（四步流程）
3. 为什么值得参与（六大原则）

**视觉节奏**: 激情 → 理念 → 方法 → 行动 → 信任 → 行动召唤

### 2.2 页面结构 (8 个 Section)

| # | Section | Class | 核心内容 | 动画 |
|---|---------|-------|----------|------|
| 1 | **Hero** | `.hero` | 品牌语 + CTA 双按钮 + 统计数据 + 光晕背景 | fadeInUp/Down + heroGlow |
| 2 | **宣言区** | `.manifesto` | 品牌引用文字 + 三段信念文案 | fadeInDown |
| 3 | **四步骤** | `.how` | 深色背景, 4 列网格: 发布→组建→协作→共享 | reveal (scroll) |
| 4 | **项目浏览** | `.projects` | 搜索栏 + 分类/状态 chip + ProjectCard 网格 | — |
| 5 | **角色区域** | `.roles-section` | 5 种角色卡片 + 在线人数估算 | reveal (scroll) |
| 6 | **公社原则** | `.principles` | 6 大原则卡片 3×2 网格 + 大号数字水印 | reveal (scroll) |
| 7 | **CTA** | `.cta` | 红色全幅行动召唤 + 大号水印文字 | — |
| 8 | **Footer** | `footer` | 社交链接 (GitHub/Twitter/Discord) | — |

### 2.5 项目浏览区

**筛选工具栏**:
- 搜索框: 实时过滤 (标题 OR 描述 OR 角色名)，大小写不敏感
- 分类 chip: 6 个 (全部/app/web/game/ai/tool) + 分隔线 `|`
- 状态 chip: 4 个 (全部/招募中/进行中/已完成)
- 搜索结果计数: "找到 N 个项目"

**ProjectCard 网格**: 3列 (≤1024px→2列, ≤768px→1列)

**空状态**: `.no-results` (🔍 图标 + "没有找到匹配的项目")

### 2.6 角色区域 + 原则 + CTA + Footer

**角色区域** (`.roles-section`, bg: `var(--cream)`):
5 种角色卡片, emoji + 名称 + 在线人数估算 (基于项目数据比例计算)

**公社原则** (`.principles`):
6 张卡片 3×2 网格, 每张含:
- 大号数字水印 (01-06, `opacity: 0.08`, `font-size: 4rem`, serif)
- 标题 (serif, fw 700, fs 1.15rem)
- 描述段落 (fs 0.85rem, lh 1.8, color warm-gray)

**CTA** (`.cta`, bg: `var(--red)`):
- 全屏红色背景
- 大号水印文字 "共创 共享 共建" (`font-size: clamp(6rem,15vw,14rem)`, opacity 0.05)
- 白色 btn-white 按钮

**Footer**: 简单三链接 + 版权信息

### 2.7 滚动动效

使用 `useScrollReveal()` hook + IntersectionObserver:

```js
// 注册 .reveal 元素 (opacity: 0, translateY: 40px)
// 当进入视口 (threshold: 0.1, rootMargin: 0 0 -50px) 时添加 .visible
// visible: { opacity: 1, transform: translateY(0) }
// transition: all 0.8s cubic-bezier(0.16, 1, 0.3, 1)
```

应用到的 section: manifesto / how-step / principles / roles-section / projects-grid 内的 ProjectCard

### 3.3 精选项目样式 (`.showcase-featured`)

- `border: 2px solid var(--gold)`
- `background: linear-gradient(135deg, rgba(212,168,67,0.04), white)`
- `position: relative; overflow: hidden`
- `::before`: "⭐ 精选项目" (absolute top: 1rem, right: 1.5rm, gold, tracking 0.1em, uppercase, font-size: 0.65rem)

**普通卡片** (`.showcase-card`):
- hover: 上浮 4px + 红色边框 + 阴影

**已完成 badge**: `bg: rgba(34,139,34,0.1), color: var(--success), letter-spacing: 0.05em, fw 700, padding: 0.2rem 0.6rem`

### 3.4 ⚠️ 已知 Bug 修复

**问题**: ShowcasePage 过滤 done 项目时未适配 wrapper 格式。

**现状**:
```jsx
// ShowcasePage.jsx line 8
const doneProjects = projects.filter(p => p.status === 'done')  // ❌ 不兼容 wrapper
```

**修复方案**:
```jsx
const doneProjects = projects.filter(item => (item.project || item).status === 'done')  // ✅ 兼容
```

## 4. ProfilePage — 个人主页 `/profile/:name`

### 4.1 设计意图

ProfilePage 是用户的**公共名片**，它回答："这个人是否靠谱？他做过什么？他的能力如何？"

不同于 Dashboard (私人控制台)，ProfilePage：
- **任何人可访问** (无需登录)
- **只读展示** (无编辑功能)
- **强调社会证明** (项目 + 徽章 + 技能)

### 4.3 Hero 区对比 (Profile vs Dashboard)

| 维度 | ProfilePage | DashboardPage |
|------|-------------|---------------|
| 头像大小 | **88px** (突出) | 72px (紧凑) |
| 统计项 | XP / 项目数 / 徽章数 | XP / 参与项目 / 贡献次数 / 徽章 |
| 技能展示 | **横向进度条** + 百分比 | **3列网格卡片** + emoji + 等级 |
| 等级进度 | 有独立区块 | 荣誉栏内含 |
| 交互 | 纯展示 (点击跳转) | 可编辑 |

### 4.4 未找到用户处理

## 5. 三页面协作关系

```
                    HomePage                          ShowcasePage
                       ↓                                   ↑
              浏览项目                           筛选 done 项目
                       ↓                                   ↓
              DetailPage  ──────────────→  status='done'
                       │
                       ↓
              ProfilePage (点击发起人名)
                       
  LeaderboardPage (点击排名中的用户)
```

**导航路径**:
- HomePage → DetailPage (点击 ProjectCard)
- DetailPage → ProfilePage (点击发起人名/成员名)
- Navbar → ShowcasePage (🏆 成果展厅链接)
- Navbar → Leaderboard (排行榜链接)
- ProfilePage → DetailPage (点击参与项目)
